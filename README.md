# Search Repositories

Aplicação web para buscar e listar repositórios públicos do GitHub.

## Funcionalidades

- Busca de usuários do GitHub com sugestões em tempo real
- Listagem de repositórios com informações relevantes (nome, descrição, linguagem)
- Interface responsiva e intuitiva
- Tratamento de erros amigável

## Propósito

Projeto de estudos para práticas de desenvolvimento moderno com React, aplicando conceitos e ferramentas atuais do mercado.

## Tecnologias

- **Vite** - Build tool rápida
- **React 18** - Biblioteca de UI
- **TypeScript** - Tipagem estática
- **Zustand** - Gerenciamento de estado
- **Zod** - Validação de dados
- **Biome** - Lint e formatação
- **Styled Components** - Estilização
- **React Router** - Roteamento

## Deploy

Acesse o projeto em: https://search-repositories.vercel.app/

## Como executar

```bash
# Instalar dependências
npm install

# Iniciar desenvolvimento
npm run dev

# Build produção
npm run build

# Verificar lint
npm run lint

# Formatar código
npm run format
```
